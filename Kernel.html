- (nil) close

Closes an open console session by raising a SystemExit exception. This is particularly useful when the context is bound to an object which has its own exit method defined (such as SceneManager).

Returns:

  • (nil) —

    if the console is disabled

Raises:

  • (SystemExit) —

    if the console is enabled
